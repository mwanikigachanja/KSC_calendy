# KSC Calendy - Social Media Calendar App

KSC Calendy is a simple social media calendar app built with Python and Streamlit. It allows users to schedule and manage their social media posts, view them in a calendar format, and track their scheduled posts.

## Features

- Calendar View: Display scheduled posts in a calendar format.
- Create New Post: Add new posts with content and schedule dates.
- Scheduled Posts: View a list of all scheduled posts with their details.

## Installation

1. Clone the repository:

```
git clone https://github.com/mwanikigachanja/KSC_calendy.git
```

2. Install the required dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run social_media_calendar.py
```

## Usage

- Once the app is running, navigate to different sections using the sidebar.
- Use the "Create New Post" section to add new posts by entering the post content and selecting the scheduled date.
- View scheduled posts in the "Scheduled Posts" section.
- Navigate through the calendar view to see posts scheduled for each day.
